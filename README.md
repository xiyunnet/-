## 羲云自助点餐系统

手机端的点餐系统做过很多款，以前使用H5进行开发，现在重新开发了一套微信小程序版本。  
使用这套小程序点餐，完全可以满足一般餐厅的需求，主要是功能基本简单实用，界面美观大方，提升餐厅的档次和点单率。  
由于没有餐厅资质，所以暂时没有上架小程序，需要体验、功能开发、对接、合作开发运营都可以微信联系我。  

### 特色及功能
1.简洁的界面风格，首页完成点菜功能  
2.直接点餐或者选择菜肴属性，进行功能性点餐  
3.开启多店模式  
4.微信支付支持  
5.在手机端完成餐厅设置，菜肴上传，信息更改等各种操作  
6.海报等促销功能
